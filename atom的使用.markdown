## 安装插件
file-->setting-->packages

安装在.atom-->packages  (注意是.atom)
```
C:\Users\Courage>apm install markdown-image-assistant
Installing markdown-image-assistant to C:\Users\Courage\.atom\packages done
```
如果缺少module之类的就
```
cd 插件所在的文件夹
npm install 缺少的东西
```
